🧪 Open Phantoms

Reproducible research — 3D‑printable, clinically validated, open‑source

All phantoms are developed as open‑science projects. You can reproduce them using the provided STL files and protocols from our papers. For ready‑to‑use phantoms, please visit our technology partner Moscow Center for Diagnstics and Telemedicine.

🧽 Spongy Ultrasound Training Phantom

3D‑printed porous medium (156 µm fibers) with anechoic cysts (4–8 mm). Mimics muscle tissue (1547 m/s) and breast tissue (1510 m/s). Withstands up to 300 punctures.
🔬 Open-source / Not for sale — reproduce yourself
  • 📁 STL files + print task: Download from ResearchGate →
  • 📄 Paper: Grebennikova et al., Phys Eng Sci Med, 2026 →
  • 💻 Custom software (C++): Generate your own spongy structures →
  • ⚙️ Printer: LCD stereolithography (e.g., Phrozen Shuffle 4K) · 50 µm layers · 3h 23min print time
📢 Note

This phantom is not for sale. It is provided as an open‑source resource for the research and education community. Everyone can reproduce it using the public STL files and software.

📌 Note: The raw RF dataset for Transcranial Phantom can be requested via email. For the Doppler Twinkling Artifact dataset (10.5 GB), please visit our Open Datasets section on the main page.
