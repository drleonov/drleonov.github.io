🖥️ RUFUS

RF Ultrasound Framework for Universal Simulations

RUFUS is an open‑source C++ platform for ultrasound research on real radio‑frequency (RF) data. It reads raw data from Sonomed‑500 and Verasonics scanners and provides a complete processing pipeline: B‑mode, color flow imaging (CFM), power Doppler, tissue Doppler, synthetic aperture focusing with aberration correction, and compression elastography. The source code is freely available under the MIT license.

🔗 GitHub: moscow-ultrasound-lab/rufus  |  📦 DOI: 10.6084/m9.figshare.18495170.v1  |  📄 Medical Physics 2024 paper

⚙️ Key features

🔌 Multi‑scanner RF input

Supports raw RF data from Sonomed‑500 (production) and Verasonics (development). Extensible binary format interface for other scanners.

Sonomed‑500 Verasonics 40 MHz sampling

🔬 B‑mode & synthetic aperture

Conventional and synthetic aperture beamforming. Aberration correction using Legendre polynomial approximation (up to 3.08× peak intensity recovery).

real‑time B‑mode (117 fps) SA focusing

🩸 Color flow imaging (CFM)

Complete CFM pipeline: clutter filtering (polynomial, SVD), masking (91.6% diagnostic efficiency), velocity estimation via autocorrelation.

power Doppler tissue Doppler twinkling artifact suppression

🌀 Synthetic aperture & aberration correction

Virtual point sources, post‑processing focusing, Legendre polynomial‑based phase correction. Validated on photopolymer and ex vivo temporal bone.

RMS width ↓60% peak intensity ↑3.08×

📏 Compression elastography

Strain imaging by comparing B‑mode frames before/after mechanical compression. Works with our anthropomorphic phantoms.

compatible with breast/neck phantoms

⚡ High performance (C++17)

140–170× faster than MATLAB‑based simulators (Field II, SIMUS). B‑mode conventional at 8.5 ms/frame on standard CPU.

CMake no GPU required

🚀 Quick start (example)

// Load raw RF data from Sonomed‑500
RufusReader reader;
reader.open("data/phantom_scan.dat");

// Process conventional B‑mode
BmodeImage bmode = processor.bmode(reader, params);

// Color flow imaging (CFM)
CfmParameters cfmParams;
cfmParams.clutterFilter = ClutterFilter::Polynomial;
cfmParams.maskRule = MaskRule::Combined;  // 91.6% efficiency
CfmMap cfm = processor.cfm(reader, cfmParams);

// Synthetic aperture with aberration correction
SaParameters saParams;
saParams.aberrationCorrection = true;
saParams.polynomialOrder = 4;
BmodeImage saImage = processor.syntheticAperture(reader, saParams);

// Export to DICOM or PNG
saImage.save("output.png");

📁 Full examples and build instructions are available in the GitHub repository.
